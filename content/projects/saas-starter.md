## Visão Geral

Começar um SaaS do zero é repetitivo. Configurar autenticação, pagamentos e banco de dados consome tempo precioso. O **SaaS Starter Kit** resolve isso.

## Funcionalidades

1.  **Autenticação**: Login com Google e Email (Magic Links) via Auth.js.
2.  **Pagamentos**: Integração completa com Stripe Checkout e Webhooks.
3.  **Database**: Prisma ORM com PostgreSQL (Neon DB).
4.  **UI Components**: Shadcn/ui pré-configurado.

## Como usar

```bash
git clone https://github.com/DevAirton-Jr/saas-starter
npm install
npm run dev
```
