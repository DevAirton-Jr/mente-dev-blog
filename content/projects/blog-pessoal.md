## Sobre o Projeto

Este blog foi desenvolvido com o objetivo de ser um portfólio vivo e uma plataforma para compartilhar conhecimento. Utilizando as tecnologias mais recentes do ecossistema React, ele foca em:

*   **Performance**: Utilizando Server Components para minimizar o JS no cliente.
*   **Acessibilidade**: Componentes semânticos e navegáveis.
*   **Design**: Uma interface limpa, porém com personalidade "dev".

## Stack Tecnológica

*   **Next.js 15 (App Router)**
*   **Tailwind CSS v4**
*   **Framer Motion** para animações
*   **Gray-matter** para processamento de Markdown

## Desafios

Um dos principais desafios foi implementar o sistema de arquivos como CMS, garantindo type-safety e performance na geração estática das páginas.
