## Conceito

Uma interface de chat fluida e responsiva que utiliza o Vercel AI SDK para streaming de respostas.

## Arquitetura

O projeto utiliza `useChat` para gerenciar o estado da conversa e Server Actions para comunicar com a API da OpenAI, garantindo que as chaves de API nunca sejam expostas no cliente.
