O VS Code é o editor mais popular do mundo por um motivo: é extremamente extensível.

## Extensões Essenciais

*   **ESLint/Prettier**: Para manter o código limpo.
*   **GitLens**: Para ver quem quebrou o código (provavelmente você).
*   **Live Server**: Para ver mudanças em tempo real (se não usar Next.js).

## Atalhos Mágicos

`Ctrl + D`: Seleciona a próxima ocorrência.
`Alt + Up/Down`: Move a linha.

> "A ferramenta certa não faz o mestre, mas ajuda muito."
